🚀 Classificação de Instruções RISC-V

Projeto desenvolvido para a disciplina de Arquitetura e Organização de Computadores II (UNIVALI) com o objetivo de analisar, classificar e visualizar instruções RISC-V a partir de arquivos em hexadecimal ou binário.

📌 Sobre o Projeto

Este sistema lê um arquivo contendo instruções de máquina RISC-V e realiza:

🔍 Identificação do tipo da instrução
🧠 Extração dos campos internos
🎨 Visualização em interface gráfica organizada

De acordo com a especificação da atividade:

O programa deve classificar instruções e separar corretamente seus campos com base no formato
